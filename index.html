Controle de pedidos em PHP

Uma base simples para organizar producao, entregas e operacao.

O HLP Controle e um sistema web para acompanhar pedidos, produtos, categorias, usuarios e fluxo de trabalho entre producao, pronto e entregue. Esta pagina foi preparada para rodar de forma estatica no GitHub Pages.

Recursos

O que o sistema cobre hoje

O foco do projeto e operacao cotidiana: cadastrar pedidos, acompanhar status, organizar produtos e manter uma area administrativa funcional.

Operacao

Pedidos com fluxo claro

O sistema acompanha o ciclo do pedido do inicio ao fim, com listagens, detalhes e movimentacao por status.

Cadastro

Produtos e categorias

Permite manter uma base organizada de produtos e categorias para alimentar os pedidos com mais consistencia.

Acesso

Usuarios e perfis

Conta com autenticacao, sessao, perfil de usuario e area administrativa para gerenciamento interno.

Modulos

Partes principais da aplicacao

A estrutura atual combina telas em pages/, acoes em actions/ e configuracoes centrais em config/.

Interface e navegacao

  • Dashboard com visao geral
  • Listagens de pedidos
  • Telas de producao, prontos e entregues
  • Perfil e paginas administrativas

Base tecnica

  • Autenticacao com sessao em PHP
  • Persistencia em MySQL com PDO
  • Script create_tables.sql para estrutura inicial
  • Suporte a multiempresa em config/tenant.php
Instalacao local

Como usar o projeto fora do GitHub Pages

1

Configure as credenciais do banco em config/db.php conforme seu ambiente.

2

Crie o banco com create_tables.sql ou abra install.php no navegador.

3

Sirva a pasta em Apache ou XAMPP e acesse index.php para entrar no sistema.
